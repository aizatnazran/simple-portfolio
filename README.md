# Vue Tailwind Portfolio

Minimal portfolio made on VUE JS and Tailwind CSS.

## Features

- Light/dark mode toggle
- Fetched Projects from github repositories
- Mobile responsive

## Run Locally

Go to the project directory

```bash
  cd my-portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Build the project

```bash
  npm run build
```
